    Literatura i korišteni izvori

    Popis dokumentacije, vodiča i alata korištenih pri izradi ove web aplikacije. Svim je izvorima pristupljeno u lipnju 2026. godine.

    Službena dokumentacija

    1. MDN Web Docs: HTML: HyperText Markup Language. Mozilla. https://developer.mozilla.org/en-US/docs/Web/HTML
    2. MDN Web Docs: CSS: Cascading Style Sheets. Mozilla. https://developer.mozilla.org/en-US/docs/Web/CSS
    3. MDN Web Docs: JavaScript. Mozilla. https://developer.mozilla.org/en-US/docs/Web/JavaScript
    4. MDN Web Docs: Using the Fetch API (AJAX zahtjevi). Mozilla. https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API/Using_Fetch
    5. OpenJS Foundation: Express - Node.js web application framework. https://expressjs.com/
    6. OpenJS Foundation: Node.js Documentation. https://nodejs.org/docs/latest/api/
    7. Oracle / MariaDB Foundation: MySQL 8.0 Reference Manual. https://dev.mysql.com/doc/refman/8.0/en/

    Vodiči i tutorijali

    1. CSS-Tricks: A Complete Guide to Flexbox. https://css-tricks.com/snippets/css/a-guide-to-flexbox/
    2. CSS-Tricks: A Complete Guide to CSS Grid. https://css-tricks.com/snippets/css/complete-guide-grid/
    3. W3Schools: Node.js MySQL Tutorial. https://www.w3schools.com/nodejs/nodejs_mysql.asp
    4. MDN Web Docs: Client-side form validation. https://developer.mozilla.org/en-US/docs/Learn/Forms/Form_validation
    5. MDN Web Docs: Responsive design. https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Responsive_Design

    1. express - web okvir za Node.js poslužitelj. https://www.npmjs.com/package/express
    2. mysql2 - upravljački program za spajanje na MySQL bazu podataka. https://www.npmjs.com/package/mysql2
    3. express-session - upravljanje korisničkim sesijama (prijava). https://www.npmjs.com/package/express-session
    4. bcryptjs - sigurno spremanje lozinki (hashiranje). https://www.npmjs.com/package/bcryptjs

    2. Microsoft: Visual Studio Code - uređivač programskog koda. https://code.visualstudio.com/
    3. BoardGameGeek - podaci o društvenim igrama (broj igrača, trajanje, izdavač, godina). https://boardgamegeek.com/
